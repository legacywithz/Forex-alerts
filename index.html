How To Use This App
1. Watch your chart on TradingView
2. Price hits your level → tap 15m or 1H touch button
3. See a no-wick candle → tap NO WICK
4. Previous big move at level → tap PREV MOVE
5. See a sweep beyond level → tap MANIP
6. App auto-grades A / B / C and fires alert 🔔

✓ Auto: Live prices, manipulation detection, grading, targets
✋ Manual: Touch count, no-wick, prev move confirmation
Grade Reference
🏆 A WITH TREND — All 6 criteria · Full size · TP1 TP2 TP3
⚡ A COUNTER TREND — All 6 · 50% size · TP1 focus
🥈 B SETUP — 4-5 criteria · 75% size · TP1 TP2
🥉 C SETUP — 3 or fewer · 50% size · TP1 only
